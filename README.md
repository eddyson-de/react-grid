# react-grid
Customizable data grid for React.js
### Example

[Basic example on JSFiddle](https://jsfiddle.net/dhfsk/2q1vh796/11/)

### Develop / test

Install dependencies:


```
> $ npm install
```

Start the webpack dev server:

```
> $ npm run dev
```

Open browser `http://localhost:8080/`
